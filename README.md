Multidisciplinary Optimization of Solid Rocket Motors: Theory, Methods, and Python Implementations

A Practical Guide for Working Professionals and Graduate Students

By Troy Altus
March 2026


This ebook provides a rigorous, hands-on treatment of multidisciplinary design optimization (MDO) applied to solid rocket motors (SRMs). It assumes familiarity with aerospace propulsion, numerical methods, and Python (NumPy/SciPy level). Each chapter balances theory, disciplinary models, MDO formulations, and ready-to-run Python examples (tested with NumPy, SciPy, and open-source SRM libraries where noted). Code is presented as self-contained snippets or full modules that readers can extend.
